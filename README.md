A git pre-push hook to stop code from being pushed late at night.

# Installation
```
cd mygitproject/
curl --create-dirs -o .git/hooks/pre-push https://git.sr.ht/~smlavine/pre-push/blob/master/pre-push
chmod +x .git/hooks/pre-push
```
